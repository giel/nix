Installatie NixOS

    mkdir ~/git
    cd ~/git
    mkdir -p backup/1-configs
    cd backup
    sudo cp /etc/nixos/* 1-configs/

 Ga verder als `su`
 
    sudo su
    cd /etc/nixos
    
Check configuratie:

    nixos-rebuild test
    
Edit `configuration.nix`

Wijzig hostname van `nixos` naar goede naam.

Voeg de volgende packages toe:

- delta
- git
- git-lfs
- neovim
- vim
- wget

Check configuration & switch:


    nixos-rebuild test
    nixos-rebuild switch
    
Terug naar normale user `ctl-D` in dir `~/git/backup`

Maak een nieuwe backup:

    mkdir 2-configs
    cp /etc/nixos/* 2-configs/

Reboot om alles te activeren, waaronder de nieuwe naam van de computer.
Hierna gaan we de configuraties overnemen vanuit de git repo's. Indien nodig copieer the ssh keys of genereer nieuwe.


    cd git
    git clone git@github.com:giel/nix.git
    git clone git@github.com:giel/inst.git
    git clone git@github.com:giel/mgpak.git
    git clone git@github.com:giel/stowfiles.git

Enable eigen neovim installatie

Voeg de nieuw computer toe in `nix` repo en copieer de 2 configuratie files

    cd ~/git/nix/hosts/
    mkdir fm1625lin
    cp /etc/nixos/* fm1625lin/

Nu verder in het **nix** repo en niet meer in *~/git/backup*. Zo worden wijzigingen gelijk meegenomen in versiebeheer. Wel eerst nog flakes activeren.
